# Check Existence of User name
  
### **URL**

  `/user/profile/check_username/`

### **Method:**
  
  `POST`
  

### **Data Params**

    
    {
        "username": "string"
    }
    
### **Success Response:**
  
  * **Code:** 200 <br />
  
 
### **Error Response:**
    
  
  * **Code:** 406 NOT ACCEPTABLE <br />
    
    
        [{'error': 'error msg'}]
